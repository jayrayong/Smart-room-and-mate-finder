ขั้นตอนการติดตั้ง Web Application บน Server
	1. ติดตั้ง XAMPP (xampp-win32-5.6.21-0-VC11-installer.exe)
	2. เปิด XAMPP Control Panel
	3. Start ในส่วนของ Apache และ MySQL
	4. นำโฟลเดอร์ที่เก็บ Source Code ของโครงงาน (ระบบค้นหาเพื่อนร่วมห้อง) ไปวางไว้ที่ C:\xampp\htdocs\
	5. ในหน้า XAMPP Control Panel ในส่วนของ MySQL กดปุ่ม admin เพื่อเข้าหน้า phpMyAdmin ด้วยลิ้ง http://localhost/phpmyadmin/  
	6. สร้าง Database โดยเลือก tab Database ใส่ชื่อเป็น project เลือกเป็น utf8_unicode_ci หลังจากนั้นกด Create
	7. เข้าไปใน tab SQL ของ Database ที่สร้าง หลังจากนั้นใส่คำสั่ง SQL จากไฟล์ project.sql ที่อยู่ภายในโฟลเดอร์ Source Code (หากไม่มี Editor สามารถติดตั้ง Sublime Text 3 ได้จากไฟล์ Sublime Text Build 3126 x64 Setup.exe)
